**College Canteen Hub**

A web-based canteen management system developed as a team project for our college minor project. The application helps users browse the menu and manage food orders efficiently.

**Features**

User Login
View Canteen Menu
Place Food Orders
Order Management
Simple and User-Friendly Interface

**Technologies Used**

HTML
CSS
JavaScript
SQL
